IMPORTANT: Database Configuration
=================================
This demo chart uses a shared PostgreSQL database for all components (Admiral, Keycloak, Temporal).
The database service has a static name: admiral-demo-postgresql

⚠️  LIMITATION: Due to the static database name, you cannot install multiple instances of this
chart in the same namespace. Each installation requires its own namespace to avoid conflicts.
